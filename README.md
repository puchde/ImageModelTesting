# News App

## 架構

此專案是使用Swift UIKit撰寫的新聞App，後端使用Swift Vapor進行撰寫，新聞內容來自於GoogleNews。
<br>後端API: [NewsAPIVapor](https://github.com/puchde/NewsAPIVapor)</br>

## 功能

- [x] 頭條新聞
- [x] 搜尋新聞
- [x] 標記新聞
- [x] 設定相關功能
- [x] Widget
- [ ] 推薦新聞通知
